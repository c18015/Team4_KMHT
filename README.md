# ガイカレまつり2018に向けチームでゲーム制作  

## ゲームタイトル  
　「からくりめいろ」  

## ゲームルール  
　制限時間内に迷路の中を探索してゴールを探し出す  
　迷路内には時計アイテムがあり、取ると残り時間が増える
